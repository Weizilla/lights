Raspberry PI Nightlight Controller

## Installation
1. Install virtualenv
2. Install local python 3: `virtualenv -p python3 venv`
3. Activate virtualenv: `. venv/bin/activate`
4. Install packages: `pip3 -r requirements.txt`
